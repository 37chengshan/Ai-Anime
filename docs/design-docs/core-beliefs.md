# 核心信念

## 产品信念

1. **社区闭环优先**：用户注册 → 发布作品 → 浏览互动 → AI 辅助 → 会员订阅，这个闭环必须先跑通
2. **AI 是能力不是外挂**：AI 功能深度集成到发帖、评论、搜索等核心场景，而不是独立的聊天框
3. **创作者价值驱动**：平台为创作者提供价值（AI 工具、付费教程、会员订阅），创作者为用户提供价值

## 技术信念

1. **简单优先**：能用单体解决的问题不用微服务，能用 Postgres 解决的问题不上 ES
2. **状态机显式化**：所有核心状态必须有显式状态机，不用"字段是否为空"替代状态
3. **幂等优先**：Webhook、支付回调、上传完成等关键操作必须幂等
4. **异步解耦**：耗时操作（审核、缩略图、embedding）必须异步处理

## 工程信念

1. **文档即代码**：架构、API、DB 设计必须与代码同步更新
2. **知识层显式化**：仓库不仅有代码层，还必须有知识层、计划层、机械约束层
3. **CI 即护栏**：通过 CI 强制执行规范，而不是依赖人工检查
