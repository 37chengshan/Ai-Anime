- apps/api/app/routers/uploads.py

</read_first>
<action>
在 uploads.py 中实现 POST /sign 端点:
1. 注入 boto3 client，使用 settings.R2_* 配置
2. 生成唯一 object_key: f"uploads/{asset_id}/{original_filename}"

5. 返回 { signed_url, asset_id, object_key }
</action>

<action>

</action>

4. 提交时调用 POST /api/v1/posts
5. 发布成功后跳转到 /post/{post_id}
6. 使用 Editorial 风格：Playfair Display 标题、无圆角、纸张色背景
