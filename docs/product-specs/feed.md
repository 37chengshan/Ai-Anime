# 作品流展示规格

## 页面

### 首页最新流

- URL: `/`
- 展示最新发布的作品
- 分页：游标分页，每页 20 条
- 支持切换：最新 / 热门 / 关注

### 热门流

- URL: `/trending`
- 展示热门作品（基于时间衰减算法）
- 分页：游标分页

### 关注流

- URL: `/following`
- 展示关注用户的作品（需登录）
- 分页：游标分页

### 标签页

- URL: `/tags/[slug]`
- 展示特定标签下的作品
- 分页：游标分页

## 作品卡片 (PostCard)

- 封面图（缩略图）
- 标题
- 作者头像 + 名字
- 标签（最多 3 个）
- 点赞数 + 收藏数
- 发布时间（相对时间）

## API

| 端点 | 说明 |
|------|------|
| `GET /api/v1/posts` | 作品列表 |
| `GET /api/v1/feed/trending` | 热门流 |
| `GET /api/v1/feed/following` | 关注流 |
| `GET /api/v1/tags/{slug}` | 标签页数据 |
