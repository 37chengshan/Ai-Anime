# AI 辅助功能规格

## 功能场景

### AI 辅助写帖子

- 入口：发帖页面的"AI 生成描述"按钮
- 输入：用户上传图片 + 主题关键词
- 输出：AI 生成的作品描述文本
- 用户可编辑后发布，或重新生成

### AI 辅助写评论

- 入口：评论输入框旁边的"AI 辅助"按钮
- 输入：作品内容 + 用户意图（赞美/讨论/提问）
- 输出：AI 生成的评论草稿
- 用户可编辑后发送，或重新生成

### 站内 AI 问答

- 入口：搜索框的"AI 问答"模式
- 输入：用户问题
- 输出：AI 回答 + 相关作品引用卡片
- 回答基于站内内容（RAG）

## 配额系统

### 配额来源

- 订阅会员：每月固定配额（方案决定）
- 额外购买：可购买额外配额包

### 配额消耗

| 场景 | 消耗 |
|------|------|
| write_post | 1 unit |
| write_comment | 1 unit |
| site_search | 3 units |

### 配额显示

- 用户可查看当前配额余额
- 每次调用显示消耗数量
- 配额不足时提示购买/订阅

## API

| 端点 | 说明 |
|------|------|
| `POST /api/v1/ai/write-post` | AI 辅助写帖子 |
| `POST /api/v1/ai/write-comment` | AI 辅助写评论 |
| `POST /api/v1/ai/site-search` | 站内 AI 问答 |
| `GET /api/v1/ai/quota` | 配额查询 |
| `GET /api/v1/ai/usage` | 使用记录 |
