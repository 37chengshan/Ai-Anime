# 后台审核台规格

## 功能

### 举报管理

- 举报列表：查看用户提交的举报
- 举报详情：举报原因 + 目标内容
- 状态流转：open → reviewing → resolved/rejected

### 内容审核

- 审核任务列表：待审核内容（图片、文字）
- 决策：通过 / 拒绝 / 人工复核
- 处置动作：隐藏内容 / 删除评论 / 封禁用户

### 用户管理

- 用户列表：查看所有用户
- 用户详情：资料 + 活动 + 状态
- 操作：封禁 / 解封

### 数据看板

- 基础数据：用户数、作品数、评论数
- AI 使用统计：调用次数、配额消耗
- 订阅统计：订阅数、收入

## API

| 端点 | 说明 |
|------|------|
| `GET /api/v1/admin/reports` | 举报列表 |
| `GET /api/v1/admin/moderation/cases` | 审核任务列表 |
| `POST /api/v1/admin/moderation/cases/{caseId}/decide` | 审核决策 |
