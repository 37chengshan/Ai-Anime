## 依赖方向

```
apps/web ──────► packages/sdk ──────► packages/contracts
    │                   │
    │                   ▼
    │           apps/api (via HTTP)
    │
    ▼
apps/api ──────► packages/contracts
    │
    ▼
apps/worker ◄── Celery Queue ── apps/api
```

**规则**：
- Web 层不直接访问数据库，通过 BFF 或 API 获取数据
- API 层通过 Service → Repository 访问数据库
- Worker 消费 API 层投递的异步任务
- packages/contracts 是前后端共享的类型层

## 运行时边界

| 运行时 | 端口 | 职责 |
|--------|------|------|
| `apps/web` | 3000 | 前台页面渲染 + BFF + Webhook 入口 |
| `apps/api` | 8000 | 核心 API + AI 编排 + 业务服务 |
| `apps/worker` | - | Celery 异步任务消费者 |
| PostgreSQL | 5432 | 主数据库 |
| Redis | 6379 | 缓存 + Celery Broker |
